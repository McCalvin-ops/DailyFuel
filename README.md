# DailyFuel
Daily Fuel 🔥 | A motivational quote generator for daily inspiration and content creation.

## How quotes are generated

- Quotes are fetched automatically from live public APIs when you load the app and when you tap **Generate Quote**.
- You do **not** need to add quotes manually every day.
- If a live quote API is temporarily unavailable, Daily Fuel automatically falls back to an internal backup quote list so the app keeps working.
